# foundry-blob-explorers

Bindings for [Blobscan](https://blobscan.com) and other blob explorer APIs, used by [Foundry](https://github.com/foundry-rs/foundry).

See also [Blobscan on GitHub](https://github.com/Blobscan/blobscan).

## Features

- `rustls`: Uses `rustls` for TLS (default).
- `openssl`: Uses `openssl` for TLS.
