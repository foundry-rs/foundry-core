# foundry-explorers

Crates for interacting with block and blob explorer APIs, used by [Foundry](https://github.com/foundry-rs/foundry).

## Crates

- [`foundry-block-explorers`](./block-explorers): Bindings for [Etherscan](https://etherscan.io) and other block explorer APIs.
- [`foundry-blob-explorers`](./blob-explorers): Bindings for [Blobscan](https://blobscan.com) and other blob explorer APIs.
